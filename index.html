👽  User Manual

What is MindOrbit?

MindOrbit is a visual mind-mapping app where your ideas orbit you like planets in a solar system. Tap any object to navigate or edit it.

Solar View — your universe

Tap ☀️ Sun to edit yourself (name, photo, emoji).

Tap a Planet to zoom into an Idea.

Tap a Satellite (small dot) to zoom into the parent Idea.

+ Add Idea creates a new planet orbiting you.

Idea View — inside a planet

Tap the center sphere to edit the Idea.

Tap a satellite to zoom into a Step.

+ Add Step creates a new satellite.

Step View — inside a satellite

Tap the center sphere to edit the Step.

Tap a module to zoom into it.

+ Add Module creates an orbiting module.

Module View

Tap the center sphere to edit the Module.

Navigation

Use the ← Back button or the breadcrumb cone (top-left) to navigate up levels. Tap any breadcrumb pill to jump directly to that level.

Personalisation

Every node (Me, Idea, Step, Module) supports a photo or emoji icon — visible directly on the 3D sphere. Upload a portrait for each family member to create a family galaxy!

Data & Backup

All data is stored locally on your device. Use Export JSON in Settings to back up, and Import JSON to restore. Export PDF creates a printable visual map.

Gestures

Drag to rotate the view. Pinch to zoom in/out. Tap any sphere to interact.
